### Description
A shortcut button to quickly toggle between your personalized color document overrides and the webpage's default appearance. Only for Firefox.

### Usage
1. Download the extension.
1. Go to **Settings > Language and Appearance > Override colors** in your Firefox menu and pick your preferred colors.
2. Click the extension icon to toggle the color override on or off.
3. When active, these settings will override the website's original color scheme.
